Purpose of this Application:
  The purpose of this Multithreaded application is to count the given word in a given website (url) and 
give the count of its occurence in the website url and the process continues to subsequent urls on that url and 
this continues till it gets finished and all the webistes inside a given website is covered.

Knowledge Gained:
1.  I got major understanding of how multithreading works and how to synchronise the shared counter variables 
which are shared thread-safe data structures. 
2. Also, I learnt how thread safe data structures works and why even thread safe data structures require synchronization.
